# Amazon RDS Database Deployment and EC2 Connectivity

## Project Overview

This project demonstrates how to deploy an Amazon RDS MySQL database and securely connect it with an Amazon EC2 instance.

## AWS Services Used

- Amazon EC2
- Amazon RDS (MySQL)
- VPC
- Security Groups
- SSH
- MySQL Client

## Steps Performed

1. Launch EC2 Instance
2. Connect using SSH
3. Install MySQL Client
4. Create Amazon RDS
5. Configure Security Group
6. Connect EC2 to RDS
7. Create Database
8. Create Table
9. Insert Data
10. View Data
